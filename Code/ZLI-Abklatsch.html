Auftrag: Responsive Webseite umsetzen

Erledigt: Anzeigen Zu erledigen: Abgabe einreichen Zu erledigen: Eine Bewertung erhalten

Abgabe: Zip mit allen Files
Gruppen: alleine oder 2er Gruppen

Sie sollen eine Webseite für ein fiktives Startup umsetzen. Werden Sie kreativ und überlegen Sie sich ein gutes Produkt. Ihre Webseite soll sowohl für Mobile als auch für den Desktop optimiert werden.

  1. Überlegen Sie sich, was Ihr Unternehmen anbieten könnte.
    z. B. Awesome Apps, Kingsize Hotdogs, Tasty Donuts, Delicious Fried Squid, etc.

  2. Erfinden Sie einen Namen und ein Logo für ihr Unternehmen:
    1. Use Dot-o-mator to create domain name suggestions.
    2. Hipster Logo Generator
    3. Business Name Generator

  3. Überlegen Sie sich, wie viele verschiedene HTML Seiten notwendig sind und welchen Inhalt Sie auf welcher Seite platzieren wollen (Sitemap). Halten Sie die Resultate in einer Excel Tabelle fest. Ihre Webseite muss über mindestens 3 Unterseiten verfügen

  4. Zeichnen Sie das Layout der Webseite auf dem Computer (Visio/PowerPoint/Paint) sowohl für den Desktop als auch für Mobile.

  5. Realisieren Sie die Seite in HTML und CSS.

  6. Überprüfen Sie die Resultate in verschiedenen Browsern und nehmen Sie wo nötig Korrekturen vor.
    Falls ihre Seite online ist, lohnt es sich der Einsatz von browsershots.

🎯 Bewertungskriterien

  • Namen und Logo für das Unternehmen vorhanden
  • Sitemap mit mindestens 3 Unterseiten in einer Excel Tabelle festgehalten
  • Layout der Website (Wireframe) für Mobile und Desktop definiert
  • Navigation ist konsistent und intuitiv
  • Website ist mit semantisch korrektem HTML geschrieben
  • Alle Bilder haben ein sprechendes und aussagekräftiges alt-Attribut
  • Jede Seite der Website hat einen korrekten Titel (<title> Attribut)
  • Formulare verfügen über korrekte Auszeichnungen, Labels und Fehlermeldungen (beispielsweise im Kontaktformular)
  • Website funktioniert auf allen Displays (Mobil und Desktop)
  • Website ist nach den gängigen W3C-Standards geschrieben. Markup Validation, CSS Validation, Unicorn - Der Einheitsvalidator des W3C
  • Projekt in einem Softwareverwaltungssystem abgelegt. Änderungen regelmässig gesichert.
  • Umfang entspricht der verfügbaren Zeit
